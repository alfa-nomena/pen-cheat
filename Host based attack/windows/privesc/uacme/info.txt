- UAC : 
    - User Account Control
    - Ensure that any change to the operating system require approval from administrator or a user that is part of the local administrators group
    - privileged user will be prompt with consent prompt and unprivileged user will be prompt with administrator credential prompt

- Bypass UAC : 
    - PREREQUISITES : 
        - Need an account that is part of the local administrators local group
    - Goal : do not be prompted the consent prompt that alert the user.
